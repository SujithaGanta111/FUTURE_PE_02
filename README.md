# AI UGC Ad Content Generator – Task 2

## 📌 Project Overview
This project is part of Future Interns Task 2 (2026), focused on AI Content Marketing using UGC Ads.

The goal is to create high-converting, human-like ad scripts that feel natural and relatable, similar to real Instagram Reels content.

## 🏢 Business Selected
Glow Beauty Salon – Hyderabad

## 🎯 Objective
To use prompt engineering techniques to generate:
- Attention-grabbing hooks
- UGC-style ad scripts
- Call-to-action lines
- Instagram captions

## 🧠 Tools Used
- ChatGPT (AI Prompt Engineering)

## ⚙️ How It Works
A structured prompt is designed to instruct AI to generate UGC-style content following:
Problem → Solution → Call-to-Action framework.

The output is natural, engaging, and ready for social media marketing.
